# weather-lab

Discord weather bot providing current conditions, daily forecasts, and weather alerts. Supports multiple locations and delivers weather updates on a schedule via Discord.

## Topics
python, discord, bot, weather, api, forecast

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/weather-lab.git
cd weather-lab

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 weather-lab.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
